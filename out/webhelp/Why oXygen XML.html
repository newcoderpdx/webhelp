Why oXygen XML?

Using DITA can be intimidating at first, even for those who have experience with markup languages like HTML. oXygen XML has several features that make authoring much easier:

  • oXygen XML provides definitions of each tag
  • End tags are inserted automatically
  • The author view toolbar is reminiscent of Microsoft Word, most users will easily recognize many functions

  • oXygen XML automatically populates some attributes such as note type

  • XML markup is very strict; oXygen XML immediately alerts the user of syntax errors
