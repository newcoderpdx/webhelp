How to create a reference topic in oXygen XML

A step-by-step guide.

You'll need a functioning copy of oXygenXML. 30-day free trials are available at oXygenXML.com .
  1. Open oXygen XML.
  2. Go the File menu.
  3. Click on New...
  4. Under Search or select a document type, go to the folder that says Framework templates.
  5. Click on the DITA folder.
  6. Choose the topic folder.
  7. Finally, click on the reference icon.
    A new reference topic will be created.
